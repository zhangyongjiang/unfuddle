Click-Click

for quick access of your favorite applications!

 

Use Case - One Click!

One click to launch your favorite application.
  1. Start Click-Click application
  2. Add shortcut. Find your favorite application for the list, click it. The application icon will appear on the right side which lists your current shortcuts added.
  3. Press BACK button to exit One Click Shortcut.
  4. Clicking the system time area in the notification/status bar on the top right corner will launch your application immediately.
* Please make sure you have one and only one shortcut defined for this use case.

Use Case - Multiple shortcuts

  1. Start Click-Click application
  2. Find your favorite application, click it. The application icon will appear on the right side which lists your current shortcut defined.
  3. Repeat step above to add more shortcuts. To remove a shortcut, you can either 1) click the shortcut icon on the right side, you can delete it in the next page. or 2) long touching the shortcut icon on the right side.
  4. Press BACK button to exit One Click Shortcut when you are done.
  5. Clicking the system time in the notification/status bar on the top right corner will list all shortcuts you added, click one to launch the application.
  6. Or, long pressing the search button will bring you all the shortcuts you defined, click one to launch the application.

Use Case - Create shortcut to show application information

Application info page is usually used to force stop, uninstall and clean data for a particular application.

To create application info shortcut for a particular application, instead of clicking the application icon, LONG TOUCHING the application icon will show you how to do it.

Change Shortcut Bar Icon

  1. Start Click-Click application
  2. Click the icon on the top right corner.
  3. In the next page, click the "Change Icon".
  4. In the next page, pick up an icon for your shortcut bar. The first icon is a special invisible icon.

Change Shortcut Bar Position

  1. Start Click-Click application
  2. Click the icon on the top right corner.
  3. In the next page, click the "Change Position".
  4. In the next page, drag the icon to the destination position.
  5. To change the bar orientation, click the "Rotate" button

Use Case - Shortcut Group

You can define multiple shortcut groups. Each shortcut group could be placed on different locations. Group could be disabled/enabled. If a group is disabled, all the shortcut defined in the group will be disabled.
  1. Start Click-Click application
  2. Click the icon on the top right corner.
  3. In the next page, click the "New Shortcuts Group" at the bottom.

Auto Execution an application at a specific time (repeatedly)

  1. Start Click-Click application
  2. Click the application icon to create a shortcut. The shortcut icon will appear on the right side. Click the shortcut icon.
  3. In the next page, set the value for filed "Auto Execution Timer Expression". Press the menu button for the timer expression format.
If you need to disable a timer, you can put it in a group and then disable the group

Click Mode - Single Click vs. Double Click

By default, single click on the shortcut group activate area will show your shortcuts. You can change it to Double-Click mode in the Menu/Settings.

Click Any and Show All

If you have more than one shortcut groups, by default, each shortcut group is independent. You can change it to Linked Group in the Menu/Settings. Linked Group means, if you click group A, all other groups will be also presented.

Speed Dial

Create a shortcut for the "Speed Dial" and then edit it to add phone numbers. You can label each if you have more than one shortcut for Speed Dial.

Speed SMS

Create a shortcut for the "Speed Messaging" and then edit it to add phone number and message body. You can label each if you have more than one shortcut for Speed Messaging.
