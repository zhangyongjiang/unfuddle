Settings

 
Portfolio(s)
A Portfolio is a group of Symbols. You can add more portfolios.
 
Data Source(s)
A Data Source, which is usually a web page, has the detailed information for a symbol.
All data sources are contributed by application users. Use them at your own risk. You can add a new data source by creating your own, or, importing shared.
Tips
  • Use long touch on the symbol to delete it from its portfolio
  • Tapping the web browser to zoom-in and zoom-out
  • Long touching web page to enable/disable full screen web browser
  • Long touching the < sign to move to the first web page.
  • Long touching the > sign to move to the last page which is the admin page.
